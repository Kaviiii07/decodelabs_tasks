# DecodeLabs Frontend Development Internship

## About This Repository

Hello! I'm S Kavitha, a Computer Science Engineering student.

This repository contains the four projects I completed during my Frontend Development Internship at DecodeLabs. Through these projects, I learned the fundamentals of web development and improved my HTML, CSS, and JavaScript skills by building real applications.

## Projects

Designed and developed my personal portfolio using HTML and CSS to showcase my profile, skills, education, and projects.

### Project 2 – Responsive Portfolio
Enhanced the portfolio by making it responsive using CSS Media Queries so it works well on desktop, tablet, and mobile devices.

### Project 3 – Interactive Portfolio
Added JavaScript features such as Dark Mode, interactive buttons, and dynamic user interactions to improve the user experience.

### Project 4 – Form Design & Validation
Created a contact form with JavaScript validation to ensure proper user input and improve form usability.
